## 特性

- 项目构建 Vite
- 前端框架 Vue3
- 编程语言 Typescript
- 路由管理 Vue-router
- 状态管理 Pinia
- CSS框架 Tailwindcss v4 + postcss 拥抱原生CSS
- 代码规范 Stylelint + ESlint + Prettier
- 提交规范 CommitLint + Husky + lint-staged
- 组件库 Element Plus 按需打包
- SVG图标以组件的形式按需引入
